# Week 7: Spring Boot REST & Microservices

Weekly content focusing on RESTful API development and microservices architecture.

*   **Repository URL:** https://github.com/dhairyagothi/Digital-Nurture-JavaFSE-Cognizant

---

## Study Topics

### 1. Spring REST with Spring Boot 3
- RESTful architecture principles
- Building REST Controllers, defining Request Mappings, and handling HTTP methods (GET, POST, PUT, DELETE)
- Request & Response processing (PathVariables, QueryParameters, RequestBody, custom headers, and status codes)
- Custom global Exception Handling in REST controllers (`@RestControllerAdvice`, `@ExceptionHandler`)
- DTO (Data Transfer Object) representations and mapping
- Input validations using Hibernate Validator annotations
- HATEOAS basics
- Actuator endpoints configuration and custom metrics
- Spring Security basics for REST (authentication, authorization, JWT tokens, CORS config)
- API documentation using Swagger/OpenAPI

### 2. Microservices with Spring Cloud
- Monolithic vs Microservices architecture (benefits and challenges)
- Service Discovery & Registration with Netflix Eureka
- API Gateway routing and filters (Spring Cloud Gateway)
- Inter-service communication patterns (Synchronous Feign clients, Asynchronous Message queues)
- Fault Tolerance and Resilience patterns (Circuit Breaker, retries, fallbacks)
- Externalized centralized configurations using Spring Cloud Config Server
- Distributed monitoring and log tracing patterns

## Weekly Tasks
- Develop REST APIs for CRUD operations with input validation.
- Implement a global exception handler for REST endpoints.
- Build a service registry and register multiple client microservices.
- Configure Spring Cloud Gateway for request routing.
- Integrate token-based JWT protection onto endpoints.
