# Git Remote Sync Hands-On Lab (HOL-5)

This repository demonstrates how to configure remotes and coordinate changes between local working repositories and remote origins.

## Objectives Covered
- Understand how local and remote repositories interact.
- Add remote aliases (`git remote add origin`).
- Push local branch configurations to remote branches (`git push`).
- Pull the latest changes from remote repositories (`git pull`).

## Simulated Architecture
- **remote_repo.git**: Bare Git repository representing the remote origin.
- **GitDemo**: Local clone/workspace synced with origin.
