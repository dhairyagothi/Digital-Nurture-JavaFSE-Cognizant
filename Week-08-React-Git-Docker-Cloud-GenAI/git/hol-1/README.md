# Git Configuration & Setup Hands-On Lab (HOL-1)

This repository contains the completed tasks for the Git hands-on lab.

## Objectives Covered
- Setting up machine-level Git configurations (username, email).
- Integrating Notepad++ as the default global core editor.
- Initializing a local repository (`GitDemo`).
- Creating, tracking, and committing files.

## Project Structure
- **GitDemo/**: The initialized Git repository.
  - **welcome.txt**: Sample text file committed to the repository.
