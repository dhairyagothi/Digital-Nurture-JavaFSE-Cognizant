# Week 8: React, Git, Docker, Cloud & GenAI

Weekly content focusing on frontend development, source control, containerization, cloud, and generative AI fundamentals.

*   **Repository URL:** https://github.com/dhairyagothi/Digital-Nurture-JavaFSE-Cognizant

---

## Study Topics

### 1. Front-end Framework - React
- Single Page Applications (SPA) concepts
- React basics (create-react-app, directory structure)
- React Components (Functional and Class components)
- State management and Hooks (`useState`, `useEffect`, `useContext`)
- Forms and user input validations
- Application debugging techniques

### 2. Platforms & DevOps
- **Git:** Version control, basic operations, branching, merging, conflict resolution
- **CI/CD:** Pipelines and continuous integration concepts
- **Docker:** Containers vs Virtual Machines, writing Dockerfiles, and Docker Compose configuration
- **Cloud Fundamentals:** Core cloud models (IaaS, PaaS, SaaS) and deployment overviews (AWS, Azure)
- **Generative AI:** Introduction to GenAI, Large Language Models (LLMs), prompt engineering basics, and tools

---

## Weekly Tasks
- Develop a stateful React application managing forms with input verification.
- Run Git operations demonstrating conflict resolution on a branch.
- Write a Dockerfile to package a Spring Boot application and execute container runs.
