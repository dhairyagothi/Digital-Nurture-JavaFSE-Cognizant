# Week 2: PL/SQL Programming

Weekly content focusing on procedural SQL logic.

*   **Repository URL:** https://github.com/dhairyagothi/Digital-Nurture-JavaFSE-Cognizant

---

## Study Topics

### 1. Basic Syntax & Environment
- PL/SQL Block Structure (Anonymous vs Named Blocks)
- Declaring Variables and Data Types
- Assigning values

### 2. Control Structures
- Conditional Statements (IF-THEN-ELSE, CASE)
- Loops (FOR, WHILE, and standard LOOP)

### 3. Exception Handling
- Predefined Exceptions
- User-Defined Exceptions
- Exception propagation

### 4. Cursors
- Implicit vs Explicit Cursors
- Cursor Operations (OPEN, FETCH, CLOSE)

### 5. Stored Procedures & Functions
- Creating and managing Stored Procedures
- Creating and managing Functions
- Parameter Modes (IN, OUT, IN OUT)

### 6. Packages & Triggers
- Package Specifications and Bodies
- Types of Triggers (BEFORE, AFTER, INSTEAD OF)
- Creating and managing Triggers
